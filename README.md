Yes, this CKAN preset has seven full systems.(eight with stock)
This took around half a week to sort out all the mods.

CKAN MUST BE INSTALLED. Link:https://github.com/KSP-CKAN/CKAN/releases. When CKAN is launched, click file, Install from .ckan, go inside the folder this is in, and click open on optimized or regular. Then, click apply. All of these fantastic mods will then start applying to your ksp folder.

Disclaimer for volumetrics: If you install EVE Volumetrics, make sure you also drag in its bundled folders.
If you skip them, EVE’s 2D cloud layers will override the volumetric ones.

There is an optimized version of this mod pack. If your pc has low ram, GPU, or whatever, I recommend to use that version.

WARNING: This mod is very heavily dependent on your computer's statistics. Even with my RTX 5060, it still lags when loading in some planets with volumetric clouds. Promised worlds DO NOT have volumetric configs. Do not be surprised if you only see Environmental Visual Enhancement's clouds.

Version:1.12.5 ONLY.

All the mods are available on CKAN once the file has been imported into CKAN.

Disclaimer: The optimized version has limited visual mods and cosmic serenity deinstalled.
